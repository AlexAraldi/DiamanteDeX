# DiamanteDeX

![](https://i.imgur.com/UQCszrI.gif)

## Projeto
Desenvolvido durante o curso FullStack Academia do Programador 2025.

## Funcionalidade

Quando usúario digita no prompt de comando um valor ímpar, o programa desenha um diamante na tela de acordo com o valor base da linha e coluna central.
Isso refere ao tamanho do diamante.
